Principais formatações

Negrito ou Destaque

Nessa frase temos um termo em Negrito usando a tag b (não semantica)

Itálico ou Ênfase

Nesta frase, temos um termo em itálico usando a tag i (não semantica)

Texto marcado

Podemos criar também um texto marcado usando a tag mark

Texto grande e pequeno

Estamos criando um texto grande (que não serve mais) e um texto pequeno (que serve para as linhas pequenas nos contratos) neste parágrafo

Texto deletado

Podemos marcar um texto excluido (riscado) para indicar que ele deve ser lido mas não considerado com a tag del

Texto inserido

Podemos marcar um texto como inserido (sublinhado) para dar uma ênfase e indicar que ele foi adicionado depois, com a tag ins

Existe também o sublinhado com a tag u que é não semântica

Texto sobrescrito

Para inserir coisa do tipo x85 + y, usamos a tag sup

Texto subscrito

Para coisas como H2O usamos a tag sub
